Language.PlutusCore.Interpreter.LMachine

Description

The L machine A lazy machine based on the L machine of Friedman et al. [Improving the Lazy Krivine Machine] For more details see the document in plutusdocsfomega/lazy-machine The code here's closely based on the CEK machine implementation.
